# 播放器功能清单

## 功能列表

### 1. 播放/暂停按钮 ✅
- **位置**: 迷你播放器和展开播放器
- **功能**: 点击切换播放/暂停状态
- **实现**: `handleTogglePlay()` 函数
- **特性**:
  - 发送 `MSG.TOGGLE` 消息
  - 500ms 后检查播放状态，如果未播放则触发 `play-resume` 事件
  - 支持从暂停状态恢复播放

### 2. 上一首/下一首按钮 ✅
- **位置**: 展开播放器
- **功能**: 切换到播放列表中的上一首/下一首
- **实现**: `playAdjacentEpisode(step)` 函数
- **特性**:
  - 自动加载播放列表（如果为空）
  - 查找当前剧集在播放列表中的位置
  - 处理边界情况（无播放列表、不在播放列表中、无上一首/下一首）
  - 调用 `handlePlaylistClick()` 播放目标剧集

### 3. 快退30秒/快进30秒按钮 ✅
- **位置**: 展开播放器
- **功能**: 快退/快进30秒
- **实现**: 直接发送 `MSG.SEEK` 消息
- **特性**:
  - 快退时确保时间不小于0
  - 快进时允许超过当前进度（由播放器处理）

### 4. 播放列表按钮 ✅
- **位置**: 展开播放器
- **功能**: 显示/隐藏播放列表面板
- **实现**: `togglePlaylist()` 和 `closePlaylist()` 函数
- **特性**:
  - 打开时加载播放列表数据
  - 支持动画效果（CSS transition）
  - 点击遮罩层或关闭按钮关闭
  - 点击剧集项播放对应剧集

### 5. 收藏按钮 ✅
- **位置**: 展开播放器
- **功能**: 添加/取消收藏当前播放的剧集
- **实现**: `toggleFavorite()` 函数
- **特性**:
  - 发送 `MSG.FAVORITE_ADD` 或 `MSG.FAVORITE_REMOVE` 消息
  - 更新心形图标颜色和填充状态
  - 错误处理和日志记录

### 6. 下载按钮 ✅
- **位置**: 展开播放器
- **功能**: 下载当前播放的剧集
- **实现**: `handleDownloadClick()` 函数
- **特性**:
  - 三种状态：idle（可下载）、downloading（下载中）、cached（已下载）
  - 下载中显示进度百分比
  - 已下载状态点击打开文件夹
  - 下载中点击取消下载
  - 轮询下载进度（1秒间隔）

### 7. 倍速按钮 ✅
- **位置**: 展开播放器
- **功能**: 循环切换播放速度
- **实现**: `cycleSpeed()` 函数
- **特性**:
  - 支持速度：0.5x, 0.75x, 1x, 1.25x, 1.5x, 2x
  - 发送 `MSG.SPEED` 消息
  - 更新按钮显示文本

### 8. 进度条 ✅
- **位置**: 展开播放器
- **功能**: 显示播放进度，点击跳转
- **实现**: `handleProgressClick()` 函数
- **特性**:
  - 显示当前时间和总时长
  - 点击进度条跳转到对应位置
  - 实时更新进度条宽度

### 9. 播客名称点击 ✅
- **位置**: 展开播放器
- **功能**: 跳转到播客详情页
- **实现**: `handleArtistClick()` 函数
- **特性**:
  - 关闭展开播放器
  - 关闭播放列表（如果打开）
  - 设置当前播客ID和Feed URL
  - 导航到播客详情页

### 10. 迷你播放器点击 ✅
- **位置**: 迷你播放器
- **功能**: 打开展开播放器
- **实现**: 直接设置 `isExpandedPlayerOpen.value = true`
- **特性**:
  - 点击迷你播放器主体区域打开展开播放器
  - 播放按钮使用 `stopPropagation()` 防止触发打开操作

## 消息流

### 播放控制消息
- `MSG.TOGGLE` - 切换播放/暂停
- `MSG.SEEK` - 跳转到指定时间
- `MSG.SPEED` - 设置播放速度
- `MSG.NEXT` - 下一首
- `MSG.PREV` - 上一首

### 播放列表消息
- `MSG.GET_PLAYLIST` - 获取播放列表
- `MSG.EXTRACT_RESULT` - 播放指定剧集

### 收藏消息
- `MSG.FAVORITE_ADD` - 添加收藏
- `MSG.FAVORITE_REMOVE` - 移除收藏
- `MSG.FAVORITE_CHECK` - 检查是否已收藏

### 下载消息
- `MSG.AUDIO_DOWNLOAD_START` - 开始下载
- `MSG.AUDIO_DOWNLOAD_CANCEL` - 取消下载
- `MSG.AUDIO_DOWNLOAD_PROGRESS` - 获取下载进度
- `MSG.AUDIO_CACHE_CHECK` - 检查缓存状态
- `MSG.AUDIO_OPEN_FOLDER` - 打开文件夹

## 状态管理

### 信号（Signals）
- `playbackState` - 播放状态
- `isExpandedPlayerOpen` - 展开播放器是否打开
- `isPlaylistOpen` - 播放列表是否打开
- `isPlaylistHidden` - 播放列表是否隐藏（用于动画）
- `playlist` - 播放列表数据
- `playlistIndex` - 当前播放索引
- `isFavorited` - 是否已收藏
- `downloadState` - 下载状态
- `downloadProgress` - 下载进度

## 错误处理

所有异步操作都使用 `try-catch` 包装，防止未处理的 Promise 拒绝：
- `handleTogglePlay()` - 播放切换
- `toggleFavorite()` - 收藏切换
- `handleDownloadClick()` - 下载操作
- `cycleSpeed()` - 倍速切换

## 动画效果

### 播放列表面板
- 打开：先移除 `hidden` 类，下一帧添加 `visible` 类
- 关闭：先移除 `visible` 类，350ms 后添加 `hidden` 类
- 使用 CSS transition 实现平滑动画

## 收藏页面功能

### 11. 视图切换按钮 ✅
- **位置**: 收藏页面标题栏右侧
- **功能**: 在树形视图和列表视图间切换
- **实现**: `toggleViewMode()` 函数
- **特性**:
  - 两个图标按钮：树形 `[≡]` 和列表 `[☰]`
  - 选中状态高亮（主色调）
  - 记住用户选择（localStorage）

### 12. 树形视图 ✅
- **位置**: 收藏页面
- **功能**: 按播客栏目分组显示收藏
- **实现**: `TreeView` 组件
- **特性**:
  - 按 `podcastId` 分组
  - 显示播客封面、名称和数量
  - 默认全部收起
  - 点击播客名称跳转详情页
  - 点击箭头展开/收起

### 13. 列表视图 ✅
- **位置**: 收藏页面
- **功能**: 简洁列表显示收藏
- **实现**: `ListView` 组件
- **特性**:
  - 简洁列表
  - 不显示播客名称
  - 只显示封面和标题

### 14. 收藏删除 ✅
- **位置**: 收藏页面
- **功能**: 取消收藏（带确认）
- **实现**: `handleRemove()` 函数
- **特性**:
  - 第一次点击显示"确认？"
  - 3秒后自动重置
  - 确认后执行删除
  - 删除动画效果

## 历史页面功能

### 15. 按日期分组显示 ✅
- **位置**: 历史页面
- **功能**: 按日期分组显示收听历史
- **实现**: `groupByDate()` 函数
- **特性**:
  - 今天、昨天、更早三个分组
  - 显示收听进度条
  - 显示已收听时长

### 16. 恢复播放 ✅
- **位置**: 历史页面
- **功能**: 从上次位置继续播放
- **实现**: `handlePlay()` 函数
- **特性**:
  - 发送 `play-resume` 事件
  - 恢复上次播放位置

### 17. 删除历史记录 ✅
- **位置**: 历史页面
- **功能**: 删除单条历史记录
- **实现**: `handleRemove()` 函数
- **特性**:
  - 删除动画效果
  - 发送 `MSG.PLAY_HISTORY_REMOVE` 消息

### 18. 清除全部历史 ✅
- **位置**: 历史页面
- **功能**: 清除所有历史记录
- **实现**: `handleClearAll()` 函数
- **特性**:
  - 发送 `MSG.PLAY_HISTORY_CLEAR` 消息
