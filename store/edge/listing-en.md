# SongshuFM - Your Podcast Assistant

A clean and elegant podcast player for searching, subscribing, and listening to your favorite podcasts.

## Key Features

- 🔍 Smart podcast search
- 📻 RSS subscription support
- 🎧 High-quality audio playback
- ⏩ Variable speed (0.5x-2x)
- 📥 Offline caching
- ☁️ WebDAV data sync
- 🌙 Dark/Light theme
- 🌏 Chinese/English UI

## Privacy First

- All data stored locally
- No account required
- Open source & transparent

## Permissions Explained

- **Storage**: Save subscriptions and playback progress
- **Unlimited Storage**: Cache audio files
- **Background Playback**: Continuous audio playback
- **Alarms**: Periodic feed update checks
- **Notifications**: Podcast update alerts
- **Downloads**: Download audio files
- **Side Panel**: Persistent playback UI
